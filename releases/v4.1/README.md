# OpenSES v4.1 Release

This release contains the original OpenSES executables with standard parameter limits.

## Contents

- **OpenSES** - Linux executable (original version with standard limits)
- **OpenSES.exe** - Windows executable (legacy version with standard limits)

## Compatibility

- Linux executable: Compatible with Linux x86-64 systems
- Windows executable: Compatible with Windows x86-64 systems

## Features

This version maintains the original parameter limits as defined in the initial codebase:
- Up to 900 sections
- Up to 600 aerodynamic and thermodynamic nodes each
- Up to 620 line segments
- Up to 406 ventilation shafts
- And other standard limits

## Usage

### Linux:
```bash
./OpenSES input_file.inp
```

### Windows:
```cmd
OpenSES.exe input_file.inp
```
